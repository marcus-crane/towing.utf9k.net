Vehicles towed in Auckland, NZ

This is a visualisation of vehicles that were towed between August 2016 to July 2019.

Click and drag to move around

Hold Shift while dragging to change view angle.

Resources used:

  • Towing data sourced from NZ Police via OIA
  • NZ Street Address (LDS)
  • Openrouteservice
  • Geofabrik OSM extract for New Zealand

Feel free to check out the map source code or the scripts used to cleanse and generate routing data.
